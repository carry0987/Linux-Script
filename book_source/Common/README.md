## Common Setting
### .bashrc
```bash
alias ll='ls -lh'
alias la='ls -a'
alias tt='sudo sh tools'

```

## Hint
If `bash: command not found`
```bash
export PATH="$PATH:/sbin:/usr/sbin:usr/local/sbin"
```

Add ssh-key for `SSH` connection
```bash
sudo vim ~/.ssh/authorized_keys
```
